## Tech Stack
- Java 17
- Spring Boot
- Maven (including Maven Wrapper for easy environment setup)
---
## How to run

Clone the repository:
git clone https://github.com/Iskcuser/HR-System.git

Navigate to the project directory:
cd HR-System

Run the application:
./mvnw spring-boot:run
